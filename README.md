# 레이싱 게임 3rd

경륜/경정 시뮬레이션 게임의 베팅 시스템 구현

## 주요 기능

- 게임 모드 선택 (경마/경정)
- 난이도 설정 (쉬움/보통/어려움)
- 다양한 베팅 타입
  - 단승식
  - 연승식
  - 복승식
  - 쌍승식
  - 삼복승식
- 실시간 배당률 확인
- 직관적인 UI/UX
  - 선수 정보 그래프
  - 스크롤 가능한 선수 목록
  - 고정된 베팅 컨트롤

## 기술 스택

- React + TypeScript
- Vite
- Zustand (상태 관리)
- CSS Modules

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
